# 💡 Life Hack or Urban Myth?

A native Android flashcard quiz app built with **Kotlin** and **Android Studio**.

---

## 📱 About the App

Life hacks and internet rumours are everywhere. Users often struggle to tell the difference between real productivity tips and viral myths. This app tests your common sense with 10 flashcard-style questions — can you spot the myth?

---

## 🎮 How to Play

1. Launch the app and press **START QUIZ**
2. Read the statement on the flashcard
3. Press **✅ HACK** if you think it is a real life hack, or **❌ MYTH** if it is an urban myth
4. Get instant feedback after each answer
5. Press **NEXT** to move on
6. At the end, see your total score and personalised feedback
7. Press **REVIEW ANSWERS** to see all questions with explanations

Features
✅ 10 Life Hack vs Urban Myth questions
✅ Loop through questions with score tracking
✅ Instant feedback per answer
✅ Personalised score feedback (Master Hacker, Great job, Keep practising!)
✅ Full answer review screen
✅ Clean dark-themed UI
✅ Logging throughout with Log.d
✅ Unit tests for quiz logic
✅ GitHub Actions CI for automated builds and tests
